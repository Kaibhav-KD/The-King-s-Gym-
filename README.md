# The King's Gym - Premium Fitness Website

This repository contains the source code for "The King's Gym," a modern, single-page website designed for a high-end fitness center. The project focuses on delivering a premium user experience through advanced animations, a sleek dark-themed design, and a compelling narrative structure.

## ✨ Features

- **Cinematic Loading Sequence:** An engaging loader animation to welcome users.
- **Advanced GSAP Animations:** Sophisticated, scroll-triggered animations that bring content to life as the user navigates the page.
- **Smooth Scrolling:** Implemented with Lenis for a fluid and seamless browsing experience.
- **Fully Responsive Design:** A layout that adapts perfectly to all screen sizes, from mobile phones to widescreen desktops.
- **Hero Section:** A stunning, full-screen hero section with choreographed text and image animations.
- **Interactive Sections:** Dedicated sections for Features, Trainers, Membership Plans, Testimonials, Photo Gallery, and Contact information.
- **Sticky Navigation:** A navbar that remains visible and accessible as the user scrolls.
- **Modern Aesthetics:** A clean, dark theme with bold typography and red accents for a powerful and professional look.

## 🚀 Getting Started

This is a static website built with HTML, CSS, and JavaScript. No build process or dependencies are required to run it.

### Prerequisites

You just need a modern web browser that supports HTML5, CSS3, and JavaScript (ES6+).

### Installation

1.  Clone the repository or download the source code.
2.  Navigate to the project directory.
3.  Open the `index.html` file directly in your web browser.

That's it! The website will run locally, though an internet connection is required to load fonts and images from external sources.

## 🛠️ Technologies Used

-   **HTML5:** For the structure and content of the website.
-   **CSS3:** For styling, layout, and responsive design, using modern features like CSS Variables.
-   **JavaScript (ES6+):** For interactivity and DOM manipulation.
-   **[GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/):** A professional-grade JavaScript animation library used for all the complex animations and scroll-triggered effects.
-   **[ScrollTrigger](https://greensock.com/scrolltrigger/):** A GSAP plugin for creating scroll-based animations.
-   **[Lenis](https://lenis.studio/):** For a smooth, high-performance scrolling experience.
-   **[Google Fonts](https://fonts.google.com/):** Used for the 'Teko' and 'Montserrat' typefaces.

## 📂 File Structure

The project is organized into three main files:

```
GYM-web-2/
├── index.html    # The main HTML document containing the structure.
├── style.css     # The stylesheet with all the CSS rules.
└── script.js     # The JavaScript file with all the animation logic.
```
